# Luxury Gold Theme Documentation

## Overview
- **Theme Name**: Luxury Gold
- **Theme Version**: 3.0.0
- **Author**: Insurio Studio
- **Design Philosophy**: High-net-worth private banking & wealth insurance inspired by Bentley and Rolex. Editorial serif typography and gold accents.

## Design Tokens & System
- **Typography**: `Cinzel` (Headings), `Plus Jakarta Sans` (Body)
- **Icon Pack**: Lineicons / Lucide Gold Accent
- **Primary Color**: `#B45309` (Prestige Gold)
- **Secondary Color**: `#D97706` (Amber Accent)
- **Background**: `#09090B` (Zinc Dark)
- **Card Background**: `#18181B` (Zinc Dark Card)

## Homepage Structure
- Gold border header with VIP concierge link
- High-net-worth hero header for private assets & yachts
- Bespoke private quote request section
