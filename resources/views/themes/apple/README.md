# Apple Insurance Theme Documentation

## Overview
- **Theme Name**: Apple Insurance
- **Theme Version**: 3.0.0
- **Author**: Insurio Studio
- **Design Philosophy**: Apple.com-inspired ultra-minimalist, high-definition layout. Large typography, zero clutter, big whitespace.

## Design Tokens & System
- **Typography**: `Inter` Variable (Weights: 400, 600, 700, 800)
- **Icon Pack**: Feather / Lucide Minimal Line Icons
- **Primary Color**: `#000000` (Pure Black)
- **Secondary Color**: `#1D1D1F` (Apple Dark Slate)
- **Background**: `#FFFFFF` (Pure White)
- **Card Background**: `#F5F5F7` (Apple Light Neutral)

## Homepage Structure
- Minimalist sticky navbar with pill CTA
- Ultra-large hero header ("L'assurance. Redéfinie.")
- Product showcase blocks with large spacing
- Minimalist footer with privacy & legal notices
