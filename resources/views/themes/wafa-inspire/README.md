# Wafa Inspire Theme Documentation

## Overview
- **Theme Name**: Wafa Inspire
- **Theme Version**: 3.0.0
- **Author**: Insurio Studio
- **Design Philosophy**: Wafa Assurance-inspired emerald green family theme. Warm, approachable, soft rounded cards.

## Design Tokens & System
- **Typography**: `Plus Jakarta Sans`
- **Icon Pack**: Lucide Soft Rounded Line Icons
- **Primary Color**: `#047857` (Emerald Green)
- **Secondary Color**: `#10B981` (Bright Emerald)
- **Background**: `#F0FDF4` (Emerald Soft Tint)
- **Card Background**: `#FFFFFF` (Pure White)

## Homepage Structure
- Friendly header with family portal access
- Warm family hero section
- Healthcare & life insurance focus grid
