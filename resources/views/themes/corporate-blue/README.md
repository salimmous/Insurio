# Corporate Blue Theme Documentation

## Overview
- **Theme Name**: Corporate Blue
- **Theme Version**: 3.0.0
- **Author**: Insurio Studio
- **Design Philosophy**: Allianz-inspired institutional corporate layout. High-solvency feel for PME and large industrial holding companies.

## Design Tokens & System
- **Typography**: `Plus Jakarta Sans`
- **Icon Pack**: Lucide Outline Line Icons (stroke 1.75)
- **Primary Color**: `#1E3A8A` (Deep Navy)
- **Secondary Color**: `#2563EB` (Royal Blue)
- **Background**: `#FFFFFF` (Pure White)
- **Card Background**: `#F8FAFC` (Slate Tint)

## Homepage Structure
- Top assistance info bar with emergency contacts
- Sticky corporate header with institutional quote button
- Hero section with ACAPS certification badge & risk simulator box
- Partner accreditations strip (AXA, RMA, Sanlam, Wafa, Zurich, Allianz)
- 6-service risk grid (Auto, Habitation, Santé, Entreprise, Voyage, Flottes)
- Institutional footer with legal notices

## Customization Boundaries
- **Editable by Agency**: Logo, texts, images, phone, address, WhatsApp, Google Maps, SEO meta tags, social links.
- **Locked by Super Admin**: Layout, component architecture, CSS grid, fonts, icon library, animation system.
